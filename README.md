# KOL Verdict Marketing

Standalone public marketing site for KOL Verdict.

- Marketing domain: `kolverdict.fun`
- App domain: `https://app.kolverdict.fun`
- Stack: Next.js App Router, TypeScript, Tailwind CSS
- Deployment target: Vercel

## Local Development

```bash
npm install
npm run dev
```

## Production Check

```bash
npm run lint
npm run typecheck
npm run build
```
